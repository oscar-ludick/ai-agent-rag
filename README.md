# Install

### Download Ollama
https://ollama.com/download

### Download Ollama3.1 Model 
https://ollama.com/library/llama3.1

### Download and install Postgres
https://www.postgresql.org/download/macosx/

#### Install
```
npm install
```

#### Migrate
```
npm run db:migrate
```

#### Run
```
npm run start
```
